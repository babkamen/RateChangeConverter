RateChangeObserver
------------
Observes USD exchange rates on timeline

[<img src="http://i.imgur.com/IquXRiR.png">](#)


Technologies:
---
**Backend:**
- Java 8
- Lombok
- Maven
- Spring 
- Hibernate
- Jackson
- Thymeleaf
- Apache POI
- Mockito, Restito
- HsqlDb

**Frontend:**
- Bootstrap 3
- Jquery
- DateRangePicker
- Chart js
- Moment js
- Pace js

### How do I run project ###
    mvn clean package
    
And deploy to tomcat
